수학적 귀납법과 반복문 불변식

수학적 귀납법 : 반복적인 구조를 갖는 명제들을 증명하는 데에 사용하는 증명 기법
ex.사다리 게임
단계 나누기 : N개의 세로줄에서부터 원하는 사다리가 될 때까지 한단계씩 가로줄을 긋는다.
첫 단계 증명 : 항상 맨 위의 선택지와 맨 아래의 선택지는 1:1 대응이 된다.
귀납 증명 : 가로줄을 그어 두 개의 세로줄을 연결하면 결과가 서로 뒤바뀌지만 1:1 대응 속성은 유지된다.

반복문 불변식 
1. 반복문 진입시에 불변식이 성립함을 보인다.
2. 반복문 내용이 시작할 때 불변식이 성립했다면, 끝날 때도 불변식이 항상 성립해야 한다.
3. 반복문 종료시에 불변식이 성립했다면, 항상 우리가 정답을 구함을 알 수 있다.
ex. 이진 탐색
int lo = -1; hi = n;
//불변식1 lo < hi
//불변식2 A[lo] < x <= A[hi] 
while(lo + 1 < hi) {
    int mid = (lo + hi) / 2;
    ...
}

lo + 1 = hi : while문이 종료할 때 lo+1>hi인데, 불변식에 의해 lo<hi가 성립하니 lo+1=hi일 수 밖에 없다.
A[lo] < x <= A[hi] : 불변식이 성립한다고 가정했으니 성립할 수 밖에 없다.
이로 인해 우리가 원하는 답은 항상 hi라는 것을 알 수 있다.

귀류법 : 반대되는 상황을 가정하고 논리를 전개하여 결론이 잘못돼었음을 찾아 내는 증명 기법
대부분 어떤 선택이 항상 최선임을 증명하고자 할 때 많이 이용됨.
ex. 책장 쌓기
각 책장의 무게 Wi와 버틸 수 있는 무게 Mi가 주어진다면, 가장 높이 책장을 몇개 쌓을 수 있는가?
책장 위로 올라간 다른 책장들의 무게의 합이 견딜 수 있는 무게를 초과하면 안된다는 의미이다.
문제를 풀기 위해서는 책장을 쌓는 순서를 결정해야 하는데, 답부터 말하자면 mi+Wi가 큰 것부터 아래에 놓아야 한다는 것이다.

이를 귀류법으로 증명하자면, 우리가 증명하고자하는 사실의 반대를 가정해야한다.
어떤 입력의 최적해를 구했는데 Mi+Wi가 더 큰 책장 A가 더 작은 책장 B 위에 올라간 형태라고 가정한다.
이때 A와 B의 위치를 항상 바꿀 수 있음을 증명하면 우리가 원하는 결과를 도출할 수 있음.
MA + WA > MB + WB 를 좌변을 이항해 MA > MB + WB - WA로 표현할 수 있다.
A 상자위에 물건들이 올려져 있을 때 이 때 무게를 X라고 한다면,
A가 B 위에 올라가 있는 상태로 가정했으니 MB >= WA+X이고, MA > MB + WB - WA >= (WA+X)+WB-WA 가 된다. 

식을 계산하면 MA > X+WB 만 남는데, 이는 A는 B와 나머지 상자(X)들을 지탱할 수 있음을 나타내게 된다.


비둘기집 원리 : 10마리의 비둘기가 9개의 비둘기 집에 들어갔다면 2마리 이상이 들어간 비둘기집이 반드시 하나 있기 마련이다.
너무 당연한 이야기이지만, 머리털 문제에 적용해보면 서울 인구가 천만명이라고 가정했을 때 사람의 머리털이 평균적으로 10만 가닥 있는데, 넉넉잡아 100만 가닥이라고 해도
"머리털의 개수가 정확히 같은 두 사람이 반드시 존재한다."라는 사실을 금방 유추해낼 수 있다.

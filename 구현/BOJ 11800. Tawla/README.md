# [Bronze III] 11800: Tawla
[문제 링크](https://www.acmicpc.net/problem/11800)  

### 알고리즘 분류 :  
구현

### 시간 제한 :  
1 초   

### 문제 설명 :  
상근이와 창영이는 중동의 주사위 게임인 "Tawla"를 즐기고 있다. 게임에서 두 주사위를 던졌을 때, 나온 눈의 조합에 따라 부르는 별칭이 다르다.

1. 두 주사위의 눈이 다를 때: 큰 숫자를 먼저, 작은 숫자를 나중에 부른다.
    * 1: Yakk, 2: Doh, 3: Seh, 4: Ghar, 5: Bang, 6: Sheesh
    * 예외: 주사위 눈이 6과 5인 경우 "Sheesh Beesh"라고 부른다.
2. 두 주사위의 눈이 같을 때:
    * 1-1: Habb Yakk
    * 2-2: Dobara
    * 3-3: Dousa
    * 4-4: Dorgy
    * 5-5: Dabash
    * 6-6: Dosh

두 주사위의 눈이 주어졌을 때, 상근이와 창영이가 외쳐야 할 문장을 출력하시오.  

### 입력 :   
첫 줄에는 전체 테스트 개수인 $T$가 주어진다. ($1 \le T \le 100$)
그 다음 줄부터는 각 테스트에 주어지는 두 수 $a, b$가 주어진다. ($1 \le a, b \le 6$)  

### 출력 :   
각 테스트마다 상근이와 창영이가 외쳐야 할 문장을 "Case n:" ("" 제외) 을 포함하여 출력하시오.
